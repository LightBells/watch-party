# Suggested Commands
- `npm install` — install/update project dependencies.
- `npm start` — launch Express/Socket.IO server serving `public/` demo page.
- `npm run dev` — run backend under `nodemon` for live reload during development.
- `npm run lint` — run ESLint across `.js/.ts` sources.
- `npm test` — execute Jest test suite.
- `lsof -ti:3000 | xargs kill -9` — free the default dev port if already in use.
- Chrome: `chrome://extensions/` → enable developer mode → “Load unpacked” → select `extension/` for local testing.
